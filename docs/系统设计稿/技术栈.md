# iKindle 技术栈文档

## 项目概述
iKindle 是一个基于现代技术栈开发的电子书阅读平台，采用前后端分离架构。

## 后端技术栈

### 开发语言
- **Java**: 主要开发语言
- **JDK版本**: JDK 21.0.2 (LTS版本，提供长期支持)

### 核心框架
- **Spring Boot**: 3.2.3 (最新稳定版本)
  - 提供快速开发能力
  - 自动配置和约定优于配置
  - 内嵌服务器，便于部署

### 安全框架
- **Apache Shiro**: 1.13.0 (最新稳定版本)
  - 强大的Java安全框架
  - 提供认证、授权、会话管理、加密等功能
  - 简单易用，灵活配置
  - 替代Spring Security作为主要安全框架

### 数据访问层
- **Spring Data JPA**: 3.2.3 (与Spring Boot版本一致)
  - 简化数据库操作
  - 提供Repository模式
  - 支持PostgreSQL数据库
- **QueryDSL**: 5.0.0 (最新稳定版本)
  - 编译时类型检查
  - 动态查询构建
  - 与JPA无缝集成

### 缓存与消息队列
- **Redis**: 7.2.4 (最新稳定版本)
  - 高性能内存数据库
  - 支持多种数据结构
  - 实现生产者-消费者消息队列模式

### 分布式锁
- **Redisson**: 3.27.1 (最新稳定版本)
  - 基于Redis的分布式锁实现
  - 支持多种锁类型
  - 自动续期机制

### 文件存储
- **阿里云OSS**: 最新SDK版本
  - 高可用、高可靠的云存储
  - 支持CDN加速
  - 适合存储电子书文件

### 工具库
- **Hutool**: 5.8.25 (最新稳定版本)
  - 丰富的工具方法
  - 提高开发效率
  - 减少重复代码

### 代码生成工具
- **Lombok**: 1.18.30 (最新稳定版本)
  - 通过注解简化Java代码
  - 自动生成getter、setter、toString等方法
  - 减少样板代码，提高开发效率
- **MapStruct**: 1.5.5.Final (最新稳定版本)
  - Java Bean映射工具
  - 通过注解处理器生成类型安全的映射代码
  - 实现Entity与DTO之间的自动转换

### 数据库
- **PostgreSQL**: 16.2 (最新稳定版本)
  - 强大的开源关系型数据库
  - 支持JSON、全文搜索等高级特性
  - 适合复杂查询和数据分析

## 前端技术栈

### 核心框架
- **React**: 18.2.0 (最新稳定版本)
  - 组件化开发
  - 虚拟DOM
  - 丰富的生态系统

### UI组件库
- **Ant Design (antd)**: 5.15.1 (最新稳定版本)
  - 企业级UI设计语言
  - 丰富的组件库
  - 完善的文档和社区支持

- **Ant Design Mobile (antd-mobile)**: 5.34.0 (最新稳定版本)
  - 专为移动端设计
  - 支持触摸操作
  - 响应式设计

### 构建工具
- **Node.js**: 20.11.1 LTS (长期支持版本)
- **npm**: 10.2.4 或 **yarn**: 1.22.22

## 技术架构特点

### 后端架构
1. **微服务友好**: Spring Boot便于微服务拆分
2. **高性能**: Redis缓存提升系统性能
3. **可扩展**: 分布式锁支持高并发场景
4. **云原生**: 支持容器化部署
5. **安全可靠**: Apache Shiro提供认证授权功能

### 前端架构
1. **响应式设计**: 支持多端适配
2. **组件化**: 提高代码复用性
3. **用户体验**: 专业的UI组件库

## 技术选型优势

### 后端优势
- **成熟稳定**: Spring Boot生态成熟，社区活跃
- **开发效率**: JPA + QueryDSL提供高效的数据库操作
- **性能优化**: Redis缓存和消息队列提升系统性能
- **分布式支持**: Redisson提供分布式锁能力
- **代码简化**: Lombok减少样板代码，MapStruct实现安全的Bean映射
- **安全框架**: Apache Shiro提供简洁易用的安全控制

### 前端优势
- **组件丰富**: Ant Design提供完整的UI组件体系
- **移动适配**: Ant Design Mobile专为移动端优化
- **开发效率**: React生态丰富，开发工具完善
- **用户体验**: 专业的UI设计，提升用户满意度
